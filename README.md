# FILM-DATABSE-CLI-APPLICATION-
#  Overview
A terminal-based Python application that allows users to search movies and fetch detailed movie information using the OMDb API.

---

## Features
- Search movies by title
- Fetch detailed movie information using IMDb ID
- Display rating, genre, and plot
- Simple CLI interface

---

# Tech Stack
- Python
- Requests library
- OMDb API

---

# How to Run

## 1. Install dependencies### 2. Run application---

##  API Setup
- Get API key from https://www.omdbapi.com/apikey.aspx
- Replace `YOUR_API_KEY` in `api.py`
